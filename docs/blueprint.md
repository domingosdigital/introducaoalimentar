# **App Name**: Primeiras Mordidas

## Core Features:

- Daily Food Suggestion: Suggests a food item appropriate for the baby's age, updated daily.
- Texture Indication: Indicates the recommended food texture (pureed, mashed, small pieces) for the suggested food.
- Quick Recipes: Provides a list of quick and easy recipes suitable for babies.
- Food Reaction Tracking: Tracks food items already introduced to the baby, with options to mark reactions ('Liked', 'Disliked', 'Strange Reaction').
- Timeline History: Displays a simple timeline showing foods offered each day.
- Quick Guide: Offers quick, clear information about when to start introducing solids, recommended textures per phase, and basic care tips.
- Firebase Integration: Uses Firebase for simple login, baby profile creation, and storing records with 'usuarios', 'bebes', and 'historicoAlimentos' collections.

## Style Guidelines:

- Primary color: Soft peach (#FFCBA4) for a warm and welcoming feel, evoking feelings of nurturing and gentleness.
- Background color: Light beige (#F5F5DC), providing a gentle backdrop that doesn't overwhelm.
- Accent color: Muted teal (#70C1B3), offering a calming and trustworthy presence.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look with warmth for an approachable and friendly feel.
- Use simple, hand-drawn style icons representing different food groups and reactions.
- Clean, uncluttered layout with ample white space to avoid overwhelming new mothers.
- Gentle animations, such as a subtle fade-in effect when new content loads, to provide a smooth user experience.
